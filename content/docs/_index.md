Welcome to the **official Kaoto documentation**! 

Kaoto is a visual designer for Apache Camel integrations that helps you create, edit, and deploy integration flows with ease.

This documentation is organized into several sections that you can access from the side menu:

- **[Installation](installation/)** - Get started by installing Kaoto in your environment
- **[Quickstart](quickstart/)** - Learn the basics with a quick tutorial
- **[User Manual](manual/)** - Step-by-step instructions for the Kaoto designer
